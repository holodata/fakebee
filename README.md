# Fake Bee

Fake Youtube Live event producer.

## Usage

```bash
go build .
fakebee start
```

## Worker hierarchy

Each `EventWorker` produces one kind of **events** to one specific **Kafka topic**.

```mermaid
erDiagram
		Queen ||--|{ EventWorker: manages

		EventWorker {
			string Topic
			int Interval
			int BacklogSize
			string OriginChannelId
			string OriginVideoId
		}
```
